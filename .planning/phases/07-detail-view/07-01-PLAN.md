  <action>
Add a new helper method `causale_bem_modifier` to `causali_helper.rb` that returns the BEM modifier class (without the block name) for use with section headers.

This complements the existing `documento_causale_modifier` in documenti_helper.rb but returns ONLY the modifier part, making it usable for section elements that inherit from a parent .documento block.

```ruby
# Returns BEM modifier suffix based on causale
# e.g., "ordine-entrata", "vendita-uscita", ""
# Use with class_names: class_names("documento-detail__section", "documento--#{causale_bem_modifier(causale)}")
def causale_bem_modifier(causale)
  return "" unless causale

  tipo = causale.tipo_movimento
  mov = causale.movimento

  return "" if tipo.blank? || mov.blank?

  "#{tipo}-#{mov}"
end
```

Add this method after the existing `causale_border_class` method.
